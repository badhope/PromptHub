#### 🟢 复现与修复建议

**E001 & E003 - setState in Effect 修复：**

```tsx
// 修复前 (有问题)
useEffect(() => {
  const currentIndex = navItems.findIndex(/* ... */);
  setActiveIndex(currentIndex >= 0 ? currentIndex : 0);  // ❌ 同步setState
}, [pathname]);

// 修复后
const activeIndex = useMemo(() => {
  const currentIndex = navItems.findIndex(/* ... */);
  return currentIndex >= 0 ? currentIndex : 0;
}, [pathname]);  // ✅ 使用 useMemo
```
