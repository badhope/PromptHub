# Mobile Skills - 设计规范文档

## 1. 设计风格 (Design Style)

### 核心美学：Glassmorphism 玻璃态设计 + Tech 科技感

**关键特征：**
- **透明层叠**：半透明背景与模糊效果，创造深度和层次感
- **玻璃质感**：磨砂玻璃效果，背景内容透出但模糊处理
- **流动渐变**：柔和的色彩过渡，营造科技未来感
- **微妙阴影**：多层次阴影增强立体感和深度
- **动态元素**：流畅的动画过渡，提升交互体验
- **现代字体**：清晰易读的无衬线字体，科技感十足

**设计原则：**
- 保持视觉层次清晰，重要内容突出
- 使用透明度和模糊创造空间感
- 动画流畅自然，不干扰内容阅读
- 色彩对比度符合 WCAG AA 标准
- 响应式设计，适配各种设备

## 5. LOGO 设计 (Logo Concept)

### LOGO 规范

```

字体：Inter Bold
颜色：#FFFFFF
效果：玻璃态背景 + 发光效果
```

### 动态效果

```
入场动画：
  - 淡入：opacity 0 → 1 (0.6s)
  - 缩放：scale 0.9 → 1 (0.6s)
  - 缓动函数：cubic-bezier(0.16, 1, 0.3, 1)

悬停效果：
  - 发光增强：box-shadow 增强
  - 轻微上浮：translateY(-2px)
  - 过渡时间：0.3s

持续动画：
  - 背景光晕缓慢脉动
  - 边框光效流动
  - 频率：3-5秒循环
```

### LOGO 尺寸

```
桌面端：高度 80px
平板端：高度 60px
移动端：高度 48px

最小尺寸：高度 32px
最大尺寸：高度 120px
```

### LOGO 位置

```
主界面：垂直居中，水平居中
导航栏：左侧，垂直居中
```

## 9. 动画规范 (Animation Guidelines)

### 过渡时间

```
快速：0.15s (状态变化)
标准：0.3s (常规交互)
慢速：0.6s (入场动画)
```

### 缓动函数

```
标准缓动：cubic-bezier(0.4, 0, 0.2, 1)
入场缓动：cubic-bezier(0.16, 1, 0.3, 1)
弹跳效果：cubic-bezier(0.68, -0.55, 0.265, 1.55)
```

### 动画类型

```
淡入淡出：opacity
缩放：scale
移动：translate
旋转：rotate
模糊：filter: blur()
```

## 11. 性能优化 (Performance)

### 图片优化

```
格式：WebP (首选), JPEG, PNG
懒加载：使用 loading="lazy"
响应式图片：使用 srcset
压缩：质量 80-85%
```

### 动画性能

```
使用 transform 和 opacity
避免 layout 和 paint
使用 will-change 提示
GPU 加速：transform: translateZ(0)
```

### 字体加载

```
策略：font-display: swap
预加载：关键字体
子集化：仅加载需要的字符
```
