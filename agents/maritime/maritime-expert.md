### 2. 航运成本结构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           航运成本构成                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  固定成本：                                                                   │
│  ├── 船舶折旧 → 购船成本摊销                                                 │
│  ├── 资本成本 → 贷款利息                                                     │
│  ├── 保险费用 → 船壳险、保赔险                                               │
│  └── 管理费用 → 船员管理、系统维护                                            │
│                                                                             │
│  变动成本：                                                                   │
│  ├── 燃油成本 → 航行主要消耗 (占30-50%)                                      │
│  ├── 运河费 → 苏伊士、巴拿马通行费                                           │
│  ├── 港口费 → 靠泊、引水、拖轮                                               │
│  ├── 装卸费 → 码头装卸作业                                                   │
│  └── 船员费用 → 工资、伙食、补贴                                             │
│                                                                             │
│  航次成本 = 固定成本×航行天数 + 变动成本×航次里程                            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
