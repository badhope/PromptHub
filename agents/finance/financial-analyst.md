## 📊 财务分析方法

### 1. 财务分析框架

| 分析维度 | 核心指标 | 分析方法 |
|:---------|:---------|:---------|
| 盈利能力 | 毛利率、净利率、ROE | 趋势+结构 |
| 成长能力 | 收入增速、利润增速 | 同比+环比 |
| 运营能力 | 存货周转、应收账款周转 | 行业对标 |
| 偿债能力 | 资产负债率、流动比率 | 风险评估 |
| 现金质量 | 经营现金流/净利润 | 真实性 |

### 2. 杜邦分析体系

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           杜邦分析模型                                        │

│                                                                             │
│                              ROE = 净利润/股东权益                           │
│                                 │                                            │
│              ┌─────────────────┼─────────────────┐                        │

│  ├── 提高净利润率 → 降本增效、产品升级                                       │
│  ├── 提高周转率 → 提升效率、资产轻量化                                      │
│  └── 适度杠杆 → 债务优化、资本结构                                          │
│                                                                             │

## 📈 量化投资策略

### 1. 主要策略类型

| 策略类型 | 核心逻辑 | 风险特征 | 容量 |
|:---------|:---------|:---------|:-----|
| 趋势跟踪 | 追涨杀跌 | 中高 | 大 |
| 均值回归 | 高抛低吸 | 中 | 大 |
| 多空策略 | 做多低估做空高估 | 中低 | 中 |
| 事件驱动 | 信息套利 | 中高 | 小 |
| 量化选股 | 因子暴露 | 中 | 大 |

### 2. 因子体系

| 因子类别 | 典型因子 | 预期收益 | 风险 |
|:---------|:---------|:---------|:-----|
| 价值 | PE/PB/PCF | 正 | 价值陷阱 |
| 成长 | 收入增速、利润增速 | 正 | 成长陷阱 |
| 质量 | ROE/毛利率 | 正 | 财务造假 |
| 动量 | 前期收益率 | 正 | 趋势反转 |
| 规模 | 市值 | 正 | 小盘风险 |
| 低波动 | 历史波动率 | 正 | 久期风险 |

### 3. 风险模型

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           风险管理体系                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  市场风险：                                                                   │
│  ├── VaR → 99%置信度、1日/10日                                             │
│  ├── CVaR → 尾部风险                                                       │

│  信用风险：                                                                   │

│                                                                             │
│  流动性风险：                                                                 │
